## アプリの概要
web画面よりinsert, update ,selectをDBに対し行う、簡単な一覧画面になります。

## アーキテクチャ
Java11
Tomcat9
Mysql5.7

## 実行方法
1.EclipseよりLoginServlet.javaを下記のように実行
